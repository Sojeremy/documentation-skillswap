# Paramètres du compte

Cette page détaille toutes les options disponibles dans les paramètres de votre compte SkillSwap.

## Accès aux paramètres

1. Cliquez sur votre avatar en haut à droite
2. Sélectionnez **"Paramètres"** ou l'icône engrenage

## Informations du profil

### Identité

| Champ | Description | Obligatoire |
|-------|-------------|-------------|
| **Prénom** | Votre prénom affiché | Oui |
| **Nom** | Votre nom de famille | Oui |
| **Email** | Adresse email (connexion) | Oui |
| **Ville** | Votre localisation | Non |

### Photo de profil

| Option | Description |
|--------|-------------|
| **Télécharger** | Ajouter une image depuis votre appareil |
| **Supprimer** | Revenir à l'avatar par défaut |

**Formats acceptés** : JPG, PNG, GIF
**Taille maximale** : 5 Mo

### Biographie

| Paramètre | Valeur |
|-----------|--------|
| **Longueur max** | 500 caractères |
| **Format** | Texte simple |

## Compétences

### Ajouter une compétence

| Champ | Options |
|-------|---------|
| **Catégorie** | Liste des catégories disponibles |
| **Compétence** | Compétences de la catégorie |
| **Niveau** | Débutant / Intermédiaire / Expert |

### Niveaux de compétence

| Niveau | Signification |
|--------|---------------|
| **Débutant** | Connaissances de base, peut initier |
| **Intermédiaire** | Bonne maîtrise des fondamentaux |
| **Expert** | Maîtrise avancée, peut aborder les sujets complexes |

## Intérêts

### Configuration

Les intérêts fonctionnent comme les compétences :

| Champ | Description |
|-------|-------------|
| **Catégorie** | Domaine qui vous intéresse |
| **Compétence** | Compétence spécifique souhaitée |

**Note** : Pas de niveau pour les intérêts (vous souhaitez apprendre).

## Sécurité

### Mot de passe

| Action | Description |
|--------|-------------|
| **Modifier** | Changer votre mot de passe actuel |

**Exigences du mot de passe** :
- Minimum 8 caractères
- Au moins une majuscule
- Au moins une minuscule
- Au moins un chiffre

### Sessions actives

| Information | Description |
|-------------|-------------|
| **Appareil** | Type d'appareil connecté |
| **Localisation** | Lieu approximatif |
| **Dernière activité** | Date/heure de dernière utilisation |

## Notifications

### Types de notifications

| Notification | Description | Par défaut |
|--------------|-------------|------------|
| **Nouveaux messages** | Quand vous recevez un message | Activé |
| **Nouvelles évaluations** | Quand vous êtes évalué | Activé |
| **Nouveaux followers** | Quand quelqu'un vous suit | Activé |

## Zone de danger

### Suppression du compte

| Action | Conséquence |
|--------|-------------|
| **Supprimer le compte** | Suppression définitive de toutes vos données |

**Données supprimées** :
- Profil et informations personnelles
- Compétences et intérêts
- Toutes les conversations
- Évaluations données et reçues
- Historique de suivi

**Processus de suppression** :
1. Cliquez sur "Supprimer mon compte"
2. Entrez votre mot de passe
3. Confirmez la suppression
4. Le compte est supprimé immédiatement

:::danger Attention
La suppression est **irréversible**. Toutes vos données seront définitivement effacées.
:::

## Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| `Ctrl/Cmd + S` | Sauvegarder les modifications |
| `Esc` | Annuler et fermer |
