# Résolution de problèmes

Vous rencontrez un problème sur SkillSwap ? Cette page regroupe les solutions aux problèmes les plus fréquents.

## Connexion et compte

### Je ne peux pas me connecter

**Symptômes** : Message "Identifiants incorrects" ou page qui ne charge pas.

**Solutions** :
1. Vérifiez votre adresse email (attention aux fautes de frappe)
2. Vérifiez que le verrouillage majuscule n'est pas activé
3. Essayez de rafraîchir la page (Ctrl+F5 ou Cmd+Shift+R)
4. Videz le cache de votre navigateur
5. Essayez avec un autre navigateur

### J'ai oublié mon mot de passe

Actuellement, la réinitialisation de mot de passe n'est pas disponible en self-service. Contactez l'équipe SkillSwap pour obtenir de l'aide.

### Mon compte a été créé mais je ne reçois pas de confirmation

L'inscription ne nécessite pas de confirmation par email. Vous pouvez vous connecter immédiatement après l'inscription.

## Profil

### Mon avatar ne s'affiche pas

**Causes possibles** :
- Le fichier est trop volumineux (max 5 Mo)
- Le format n'est pas supporté (utilisez JPG ou PNG)
- Le chargement a échoué

**Solutions** :
1. Vérifiez la taille du fichier (< 5 Mo)
2. Convertissez l'image en JPG ou PNG
3. Réessayez le téléchargement
4. Si le problème persiste, actualisez la page

### Mes compétences ne s'enregistrent pas

**Vérifiez** :
- Vous avez bien cliqué sur "Valider" ou "Ajouter"
- Vous n'avez pas atteint la limite de 5 compétences
- Vous n'essayez pas d'ajouter une compétence déjà présente

### Les modifications de mon profil ne sont pas visibles

1. Actualisez la page (F5)
2. Déconnectez-vous et reconnectez-vous
3. Videz le cache du navigateur

## Messagerie

### Je ne peux pas envoyer de message à un membre

**Condition requise** : Vous devez suivre le membre pour lui envoyer un message.

**Solution** :
1. Allez sur le profil du membre
2. Cliquez sur "Suivre"
3. Vous pouvez maintenant démarrer une conversation

### Les messages ne s'affichent pas en temps réel

**Causes possibles** :
- Problème de connexion WebSocket
- Bloqueur de publicités/scripts

**Solutions** :
1. Actualisez la page
2. Désactivez temporairement votre bloqueur de publicités
3. Essayez avec un autre navigateur

### Je ne reçois plus de notifications de messages

Les notifications dans l'application nécessitent que vous soyez connecté. Actualisez la page pour voir les nouveaux messages.

---

## Recherche

### Aucun résultat ne s'affiche

**Vérifiez** :
- L'orthographe de votre recherche
- Le filtre de catégorie (désactivez-le pour voir tous les résultats)
- Essayez des termes plus génériques

### Les résultats ne correspondent pas à ma recherche

La recherche fonctionne sur :
- Le prénom et nom des membres
- Leurs compétences

Essayez de rechercher directement le nom d'une compétence (ex: "React", "Piano").

## Évaluations

### Je ne peux pas noter un membre

**Conditions requises** :
1. Vous devez suivre ce membre
2. Ce membre doit vous suivre (suivi mutuel)
3. Vous ne l'avez pas déjà évalué

### Ma note n'apparaît pas sur le profil

Les notes sont visibles immédiatement. Si elle n'apparaît pas :
1. Actualisez la page du profil
2. Vérifiez que vous avez bien validé l'évaluation

---

## Problèmes techniques

### La page ne charge pas / Page blanche

**Solutions** :
1. Actualisez la page (F5 ou Ctrl+R)
2. Videz le cache du navigateur
3. Essayez en navigation privée
4. Vérifiez votre connexion internet
5. Essayez avec un autre navigateur

### L'application est lente

**Suggestions** :
1. Fermez les onglets inutiles
2. Videz le cache du navigateur
3. Désactivez les extensions non essentielles
4. Vérifiez votre connexion internet

### Erreur "500" ou "Erreur serveur"

Cette erreur indique un problème côté serveur.
1. Attendez quelques minutes et réessayez
2. Si le problème persiste, l'équipe technique en est probablement déjà informée

| Navigateur | Version minimale |
|------------|------------------|
| Chrome | 90+ |
| Firefox | 88+ |
| Safari | 14+ |
| Edge | 90+ |

:::caution Navigateurs non supportés
Internet Explorer n'est pas supporté. Veuillez utiliser un navigateur moderne.
:::
