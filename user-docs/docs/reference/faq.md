# Questions fréquentes

Retrouvez ici les réponses aux questions les plus posées sur SkillSwap.

## Compte et inscription

### Comment créer un compte ?

1. Cliquez sur "S'inscrire" sur la page d'accueil
2. Remplissez le formulaire (email, mot de passe, nom)
3. Validez votre email si demandé
4. Complétez votre profil

**Voir** : [Premiers pas sur SkillSwap](/tutorials/getting-started)

### J'ai oublié mon mot de passe

1. Sur la page de connexion, cliquez sur "Mot de passe oublié"
2. Entrez votre adresse email
3. Suivez le lien reçu par email
4. Créez un nouveau mot de passe

### Comment supprimer mon compte ?

1. Allez dans Paramètres
2. Section "Zone de danger"
3. Cliquez sur "Supprimer mon compte"
4. Confirmez avec votre mot de passe

**Attention** : Cette action est irréversible.

### Puis-je changer mon adresse email ?

Oui, dans les paramètres du compte, section "Informations personnelles".

## Profil et compétences

### Comment ajouter une compétence ?

1. Allez sur votre profil
2. Section "Mes compétences"
3. Cliquez sur "Ajouter"
4. Sélectionnez catégorie, compétence et niveau

**Voir** : [Ajouter vos compétences](/tutorials/add-skills)

### Quelle est la différence entre compétence et intérêt ?

| Type | Signification |
|------|---------------|
| **Compétence** | Ce que vous pouvez enseigner |
| **Intérêt** | Ce que vous voulez apprendre |

### Comment choisir mon niveau de compétence ?

- **Débutant** : Vous connaissez les bases et pouvez initier quelqu'un
- **Intermédiaire** : Vous maîtrisez bien le sujet
- **Expert** : Vous avez une maîtrise approfondie

Soyez honnête : un niveau surévalué mènera à des déceptions.

### Ma compétence n'est pas dans la liste

Vérifiez qu'elle n'existe pas sous un autre nom. Si elle manque vraiment, contactez l'équipe via le support pour suggérer son ajout.

## Recherche et contact

### Comment trouver un membre ?

Utilisez la barre de recherche pour chercher par :
- Compétence souhaitée
- Nom d'un membre
- Catégorie

**Voir** : [Rechercher des membres](/how-to/search-members)

### Pourquoi je ne trouve personne ?

Possibles raisons :
- Termes de recherche trop spécifiques
- Peu de membres avec cette compétence
- Filtres trop restrictifs

Essayez d'élargir votre recherche ou d'utiliser des synonymes.

### Comment contacter un membre ?

1. Accédez à son profil
2. Cliquez sur "Contacter"
3. Rédigez votre message

**Voir** : [Envoyer un message](/how-to/send-message)

### Puis-je contacter n'importe qui ?

Oui, tous les membres peuvent être contactés. Soyez respectueux et pertinent dans vos demandes.

## Échanges

### Comment se déroule un échange ?

1. Contact et discussion préalable
2. Accord sur les modalités (format, horaire)
3. Réalisation de l'échange
4. Évaluation mutuelle

**Voir** : [Votre premier échange](/tutorials/first-exchange)

### Les échanges sont-ils gratuits ?

Oui, SkillSwap est basé sur l'échange de compétences, sans transaction financière entre membres.

### Où faire les échanges ?

C'est à vous de décider avec votre partenaire :
- En présentiel (café, bibliothèque, domicile...)
- En visioconférence (Zoom, Meet, Skype...)
- Par téléphone pour certaines compétences

### Que faire si l'échange se passe mal ?

1. Communiquez clairement vos attentes
2. Si le problème persiste, clôturez l'échange
3. Laissez une évaluation honnête
4. En cas de comportement grave, signalez le membre

## Évaluations

### Quand puis-je évaluer ?

Après qu'une conversation a été clôturée, le bouton "Évaluer" apparaît.

### Puis-je modifier une évaluation ?

Non, les évaluations sont définitives une fois envoyées.

### Que faire si je reçois une mauvaise évaluation ?

- Analysez le commentaire pour vous améliorer
- Les évaluations injustes se noient parmi les bonnes
- Continuez à offrir des échanges de qualité

### Les évaluations sont-elles obligatoires ?

Non, mais fortement encouragées. Elles aident la communauté.

**Voir** : [Évaluer un membre](/how-to/rate-user)

## Suivi et favoris

### Comment suivre un membre ?

Sur son profil, cliquez sur le bouton "Suivre".

### Où voir mes membres suivis ?

Dans la section Messages, lors de la création d'une nouvelle conversation, vos suivis apparaissent en priorité.

### Le membre sait-il que je le suis ?

Oui, le membre peut voir qui le suit.

**Voir** : [Suivre des membres](/how-to/follow-members)

## Disponibilités

### À quoi servent les disponibilités ?

Elles indiquent aux autres membres quand vous êtes généralement libre pour des échanges.

### Les disponibilités sont-elles engageantes ?

Non, elles sont indicatives. Vous confirmez toujours les créneaux précis par message.

**Voir** : [Gérer ses disponibilités](/how-to/manage-availabilities)

## Problèmes techniques

### La page ne charge pas

1. Vérifiez votre connexion internet
2. Rafraîchissez la page (F5 ou Ctrl+R)
3. Videz le cache de votre navigateur
4. Essayez un autre navigateur

### Je ne reçois pas les emails

1. Vérifiez votre dossier spam
2. Ajoutez SkillSwap à vos contacts autorisés
3. Vérifiez que l'adresse email est correcte dans vos paramètres

### Bug ou suggestion

Contactez l'équipe via le formulaire de contact ou l'adresse support.
