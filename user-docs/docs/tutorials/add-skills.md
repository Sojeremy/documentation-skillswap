# Ajouter vos compétences

Les compétences sont au cœur de SkillSwap. Ce tutoriel vous explique comment bien définir ce que vous pouvez offrir et ce que vous recherchez.

## Compétences vs Intérêts

Sur SkillSwap, il y a une distinction importante :

| Type | Définition | Exemple |
|------|------------|---------|
| **Compétence** | Ce que vous pouvez enseigner | "Je sais jouer de la guitare" |
| **Intérêt** | Ce que vous voulez apprendre | "Je veux apprendre l'espagnol" |

## Ajouter une compétence

### Étape 1 : Accéder à la section

1. Allez sur **"Mon profil"**
2. Trouvez la section **"Mes compétences"**
3. Cliquez sur **"Ajouter une compétence"**

### Étape 2 : Choisir la compétence

1. Une liste de catégories s'affiche
2. Parcourez ou recherchez votre compétence
3. Sélectionnez celle qui correspond

**Catégories disponibles** :
- Langues (Anglais, Espagnol, Français...)
- Musique (Guitare, Piano, Chant...)
- Informatique (Programmation, Design, Excel...)
- Sport (Tennis, Yoga, Natation...)
- Art (Dessin, Peinture, Photographie...)
- Cuisine (Pâtisserie, Cuisine asiatique...)
- Et bien d'autres...

### Étape 3 : Indiquer votre niveau

Soyez honnête sur votre niveau :

| Niveau | Signification |
|--------|---------------|
| **Débutant** | Vous pouvez initier quelqu'un aux bases |
| **Intermédiaire** | Vous maîtrisez bien le sujet |
| **Expert** | Vous avez une excellente maîtrise |

:::warning Attention
N'exagérez pas votre niveau. Les évaluations des autres membres reflèteront la réalité de vos compétences.
:::

## Ajouter un centre d'intérêt

### Étape 1 : Accéder à la section

1. Sur votre profil, trouvez **"Mes intérêts"**
2. Cliquez sur **"Ajouter un intérêt"**

### Étape 2 : Sélectionner

1. Parcourez les catégories
2. Choisissez ce que vous souhaitez apprendre
3. Validez votre choix

## Bonnes pratiques

### Combien de compétences ajouter ?

- **Minimum** : 1 compétence (obligatoire pour être trouvé)
- **Idéal** : 3 à 5 compétences
- **Maximum** : Évitez d'en mettre trop, concentrez-vous sur vos points forts

### Comment choisir ?

Posez-vous ces questions :

1. **Est-ce que je peux vraiment aider quelqu'un ?**
   - Pas besoin d'être expert, mais capable d'enseigner les bases

2. **Est-ce que j'ai du temps à y consacrer ?**
   - Chaque compétence listée peut générer des demandes

3. **Est-ce que ça m'intéresse d'en parler ?**
   - L'échange sera plus agréable si le sujet vous passionne

### Exemples de bons profils

**Profil équilibré** :
- Compétences : Guitare (intermédiaire), Python (expert), Tennis (débutant)
- Intérêts : Espagnol, Photographie, Cuisine japonaise

**Profil spécialisé** :
- Compétences : Piano (expert), Solfège (expert)
- Intérêts : Jardinage, Bricolage

## Modifier ou supprimer

Pour modifier une compétence :

1. Cliquez sur la compétence concernée
2. Modifiez le niveau ou supprimez-la
3. Enregistrez les modifications
