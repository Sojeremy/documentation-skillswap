# Créer votre profil

Un profil complet et attractif est la clé pour trouver des partenaires d'échange. Ce tutoriel vous guide pour créer un profil qui donne envie de vous contacter.

## Pourquoi un bon profil est important ?

Les membres de SkillSwap consultent votre profil avant de vous contacter. Un profil complet :

- **Inspire confiance** : Les autres savent à qui ils ont affaire
- **Facilite les échanges** : Vos compétences sont clairement identifiées
- **Augmente votre visibilité** : Vous apparaissez dans plus de recherches

## Accéder à votre profil

1. Connectez-vous à votre compte
2. Cliquez sur votre avatar en haut à droite
3. Sélectionnez **"Mon profil"**

## Les éléments de votre profil

### Photo de profil

Une photo aide les autres membres à vous identifier :

1. Cliquez sur l'icône d'avatar
2. Sélectionnez une image depuis votre ordinateur
3. L'image est automatiquement recadrée

:::tip Conseil photo
Choisissez une photo où votre visage est bien visible. Évitez les photos de groupe ou les images trop sombres.
:::

### Informations personnelles

Complétez votre fiche :

| Information | Conseils |
|-------------|----------|
| **Prénom / Nom** | Utilisez votre vrai nom pour plus de confiance |
| **Ville** | Permet de trouver des échanges locaux |
| **Bio** | Présentez-vous en quelques phrases |

### Rédiger une bonne bio

Votre bio est votre carte de visite. Incluez :

1. **Qui vous êtes** : Votre métier, vos passions
2. **Ce que vous recherchez** : Le type d'échanges souhaités
3. **Votre motivation** : Pourquoi vous êtes sur SkillSwap

**Exemple de bio** :
> "Développeur web passionné par la musique. Je cherche à améliorer mon anglais conversationnel et à découvrir la photographie. En échange, je peux vous initier à la programmation ou au tennis que je pratique depuis 10 ans."

## Ajouter vos compétences

Les compétences sont ce que vous pouvez **enseigner** aux autres :

1. Dans la section "Mes compétences", cliquez sur **"Ajouter"**
2. Recherchez une compétence dans la liste
3. Sélectionnez votre niveau (débutant, intermédiaire, expert)
4. Validez

Vous pouvez ajouter plusieurs compétences. Concentrez-vous sur celles où vous êtes vraiment capable d'aider quelqu'un.

## Ajouter vos centres d'intérêt

Les centres d'intérêt sont ce que vous souhaitez **apprendre** :

1. Dans la section "Mes intérêts", cliquez sur **"Ajouter"**
2. Recherchez un domaine qui vous intéresse
3. Validez

Ces informations aident SkillSwap à vous suggérer des profils compatibles.

## Définir vos disponibilités

Indiquez quand vous êtes disponible pour des échanges :

1. Accédez à la section "Disponibilités"
2. Sélectionnez les créneaux (matin, après-midi, soir)
3. Choisissez les jours de la semaine

Cela aide les autres membres à savoir quand vous contacter.

## Vérifier votre profil

Avant de terminer, vérifiez que :

- [ ] Votre photo est visible et de bonne qualité
- [ ] Votre bio décrit clairement vos attentes
- [ ] Vous avez au moins une compétence ajoutée
- [ ] Vous avez au moins un centre d'intérêt
- [ ] Vos disponibilités sont à jour
