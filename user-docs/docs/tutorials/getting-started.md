# Premiers pas avec SkillSwap

Bienvenue sur SkillSwap ! Ce tutoriel vous guide pas à pas pour découvrir la plateforme et créer votre compte.

## Qu'est-ce que SkillSwap ?

SkillSwap est une plateforme d'échange de compétences entre particuliers. Le principe est simple :

- Vous avez des **compétences** à partager (cuisine, guitare, programmation...)
- Vous avez des **centres d'intérêt** que vous souhaitez développer
- SkillSwap vous met en relation avec des personnes complémentaires

**Exemple** : Vous savez jouer de la guitare et voulez apprendre l'espagnol ? Trouvez quelqu'un qui parle espagnol et souhaite apprendre la guitare !

## Créer votre compte

### Étape 1 : Accéder à l'inscription

1. Rendez-vous sur la page d'accueil de SkillSwap
2. Cliquez sur le bouton **"S'inscrire"** en haut à droite

### Étape 2 : Remplir le formulaire

Renseignez les informations suivantes :

| Champ | Description |
|-------|-------------|
| **Prénom** | Votre prénom (sera visible par les autres membres) |
| **Nom** | Votre nom de famille |
| **Email** | Votre adresse email (servira d'identifiant) |
| **Mot de passe** | Minimum 8 caractères |
| **Confirmation** | Retapez votre mot de passe |

### Étape 3 : Valider l'inscription

1. Cliquez sur **"Créer mon compte"**
2. Vous êtes automatiquement connecté
3. Vous pouvez maintenant compléter votre profil

## Découvrir l'interface

Une fois connecté, vous avez accès à plusieurs sections :

### La barre de navigation

- **Accueil** : Retour à la page principale
- **Recherche** : Trouver des membres par compétence
- **Messages** : Vos conversations avec d'autres membres
- **Mon profil** : Gérer vos informations et compétences

### La page d'accueil

La page d'accueil présente :

1. **Les catégories populaires** : Parcourez les domaines de compétences
2. **Les membres les mieux notés** : Découvrez les profils actifs
3. **Comment ça marche** : Rappel du fonctionnement en 3 étapes
