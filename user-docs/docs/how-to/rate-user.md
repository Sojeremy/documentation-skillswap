# Évaluer un membre

Les évaluations sont essentielles pour la confiance sur SkillSwap. Ce guide explique comment évaluer un membre après un échange.

## Pourquoi évaluer ?

Les évaluations permettent de :

- **Remercier** un membre pour un bon échange
- **Aider** la communauté à identifier les profils de qualité
- **Construire** votre propre réputation (les membres actifs évaluent et sont évalués)

## Quand évaluer ?

Évaluez après :
- Un échange de compétences réussi
- Une session d'apprentissage
- Une interaction significative

:::tip Conseil
Évaluez rapidement après l'échange, pendant que les détails sont frais dans votre mémoire.
:::

## Comment évaluer

### Étape 1 : Accéder à l'évaluation

Depuis une conversation clôturée :

1. Ouvrez la conversation avec le membre
2. Un bouton **"Évaluer"** apparaît si la conversation est clôturée
3. Cliquez dessus

### Étape 2 : Attribuer une note

Donnez une note de 1 à 5 étoiles :

| Note | Signification |
|------|---------------|
| ⭐ | Très insatisfait |
| ⭐⭐ | Insatisfait |
| ⭐⭐⭐ | Correct |
| ⭐⭐⭐⭐ | Satisfait |
| ⭐⭐⭐⭐⭐ | Très satisfait |

### Étape 3 : Ajouter un commentaire (optionnel)

Un commentaire aide les autres membres à comprendre votre évaluation :

**Exemple de bon commentaire** :
> "Excellent échange ! Sophie est très pédagogue et patiente. J'ai beaucoup progressé en espagnol en quelques sessions. Je recommande !"

### Étape 4 : Valider

Cliquez sur **"Envoyer l'évaluation"**.

## Critères d'une bonne évaluation

Basez votre note sur :

- **La qualité de l'échange** : Avez-vous appris quelque chose ?
- **La ponctualité** : Le membre était-il à l'heure ?
- **La communication** : Était-elle claire et agréable ?
- **L'équité** : L'échange était-il équilibré ?

## Voir les évaluations reçues

1. Allez sur **"Mon profil"**
2. Consultez la section **"Évaluations"**
3. Votre note moyenne et les commentaires sont affichés

## Règles de la communauté

- Soyez **honnête** et **constructif**
- Évitez les évaluations **punitives** ou **de représailles**
- Les évaluations **ne peuvent pas être modifiées** après envoi
