# Envoyer un message

Ce guide explique comment contacter un membre et gérer vos conversations.

## Démarrer une nouvelle conversation

### Depuis un profil

1. Accédez au profil du membre souhaité
2. Cliquez sur le bouton **"Contacter"**
3. Une nouvelle conversation s'ouvre

### Depuis la messagerie

1. Allez dans **"Messages"** (navigation)
2. Cliquez sur **"Nouvelle conversation"**
3. Sélectionnez un membre parmi vos contacts ou suivis
4. Rédigez votre message

## Rédiger un message

### Interface de messagerie

- **Zone de texte** : Tapez votre message
- **Bouton d'envoi** : Cliquez ou appuyez sur Entrée

### Bonnes pratiques

✅ **À faire** :
- Présentez-vous brièvement
- Expliquez votre demande clairement
- Posez une question pour engager la discussion

❌ **À éviter** :
- Messages trop courts ou vagues
- Demandes multiples dans un seul message
- Ton trop familier avec quelqu'un que vous ne connaissez pas

## Gérer vos conversations

### Liste des conversations

Dans la section **"Messages"**, vous voyez :
- Toutes vos conversations actives
- La date du dernier message
- Un aperçu du dernier message

### Indicateurs

| Indicateur | Signification |
|------------|---------------|
| Point bleu | Message non lu |
| Date récente | Conversation active |

### Lire une conversation

1. Cliquez sur une conversation dans la liste
2. L'historique complet s'affiche
3. Vos messages apparaissent à droite, ceux du membre à gauche

## Actions sur une conversation

### Fermer une conversation

Quand un échange est terminé :

1. Ouvrez la conversation
2. Cliquez sur les options (⋯)
3. Sélectionnez **"Clôturer"**

La conversation reste visible mais marquée comme terminée.

### Supprimer une conversation

Pour supprimer définitivement :

1. Ouvrez la conversation
2. Cliquez sur les options (⋯)
3. Sélectionnez **"Supprimer"**
4. Confirmez la suppression

:::warning Attention
La suppression est irréversible. L'historique des messages sera perdu.
:::

## Notifications

- Vous recevez une notification pour chaque nouveau message
- Les conversations non lues sont mises en évidence
