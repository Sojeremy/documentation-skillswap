# Modifier son profil

Votre profil évolue avec vous. Ce guide explique comment le mettre à jour.

## Accéder aux paramètres du profil

1. Cliquez sur votre avatar en haut à droite
2. Sélectionnez **"Mon profil"**
3. Cliquez sur **"Modifier"** ou l'icône de crayon

## Modifier vos informations

### Photo de profil

1. Cliquez sur votre photo actuelle (ou l'avatar par défaut)
2. Sélectionnez une nouvelle image
3. L'image est mise à jour automatiquement

### Informations personnelles

Vous pouvez modifier :

| Champ | Modification |
|-------|--------------|
| **Prénom** | Cliquez, modifiez, enregistrez |
| **Nom** | Cliquez, modifiez, enregistrez |
| **Ville** | Utile pour les échanges locaux |
| **Bio** | Présentez-vous aux autres membres |

### Enregistrer les modifications

Après chaque modification :
1. Cliquez sur **"Enregistrer"**
2. Un message de confirmation s'affiche

## Gérer vos compétences

### Ajouter une compétence

1. Section **"Mes compétences"**
2. Cliquez sur **"Ajouter"**
3. Sélectionnez la compétence et le niveau
4. Validez

### Modifier le niveau

1. Cliquez sur une compétence existante
2. Changez le niveau (débutant/intermédiaire/expert)
3. Enregistrez

### Supprimer une compétence

1. Cliquez sur la compétence
2. Sélectionnez **"Supprimer"**
3. Confirmez

## Gérer vos intérêts

Le processus est identique aux compétences :

1. Section **"Mes intérêts"**
2. Ajoutez, modifiez ou supprimez selon vos besoins

## Paramètres du compte

### Changer de mot de passe

1. Accédez aux **paramètres du compte** (icône engrenage)
2. Section **"Sécurité"**
3. Cliquez sur **"Changer le mot de passe"**
4. Entrez l'ancien puis le nouveau mot de passe
5. Confirmez

### Supprimer son compte

:::danger Zone de danger
La suppression du compte est irréversible.
:::

1. Paramètres du compte
2. Section **"Zone de danger"**
3. Cliquez sur **"Supprimer mon compte"**
4. Confirmez en tapant votre mot de passe

Toutes vos données seront supprimées :
- Profil et informations
- Conversations
- Évaluations données et reçues
