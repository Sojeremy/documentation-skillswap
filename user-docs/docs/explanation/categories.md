# Les catégories de compétences

Les compétences sur SkillSwap sont organisées en catégories pour faciliter la recherche et la navigation. Cette page explique cette organisation.

## Pourquoi des catégories ?

### Le problème de la diversité

SkillSwap accueille une grande variété de compétences :

- Des langues aux arts
- De l'informatique à la cuisine
- Du sport à la musique

Sans organisation, trouver une compétence spécifique serait difficile.

### La solution des catégories

Les catégories offrent :

- **Une structure claire** pour naviguer
- **Un filtrage efficace** lors des recherches
- **Une découverte** de compétences connexes

## Structure des catégories

### Organisation hiérarchique

Les compétences sont organisées en deux niveaux :

```
Catégorie
├── Compétence 1
├── Compétence 2
└── Compétence 3
```

**Exemple** :

```
Musique
├── Guitare
├── Piano
├── Chant
└── Batterie
```

### Principes de classification

Les catégories suivent plusieurs principes :

| Principe | Explication |
|----------|-------------|
| **Exclusivité** | Une compétence appartient à une seule catégorie |
| **Exhaustivité** | Toutes les compétences ont une catégorie |
| **Clarté** | Les noms sont explicites et compréhensibles |

## Les grandes catégories

### Langues

Regroupe l'apprentissage des langues étrangères :

- Anglais, Espagnol, Allemand
- Mandarin, Japonais, Arabe
- Et autres langues du monde

**Particularité** : Souvent les compétences les plus échangées.

### Informatique

Couvre le domaine du numérique :

- Programmation (Python, JavaScript, etc.)
- Design (Photoshop, Figma, etc.)
- Bureautique (Excel, Word, etc.)

**Particularité** : Compétences très demandées et variées.

### Arts

Englobe les pratiques artistiques :

- Dessin et peinture
- Photographie
- Sculpture et artisanat

**Particularité** : Échanges souvent en présentiel.

### Musique

Rassemble les pratiques musicales :

- Instruments (guitare, piano, violon...)
- Chant
- Production musicale

**Particularité** : Requiert parfois du matériel spécifique.

### Sport et bien-être

Inclut les activités physiques :

- Sports collectifs et individuels
- Yoga, méditation
- Fitness, musculation

**Particularité** : Échanges généralement en présentiel.

### Cuisine

Couvre les arts culinaires :

- Cuisine traditionnelle
- Pâtisserie
- Cuisines du monde

**Particularité** : Échanges pratiques et conviviaux.

### Artisanat

Regroupe les travaux manuels :

- Couture, tricot
- Bricolage
- Menuiserie

**Particularité** : Transmission de savoir-faire traditionnels.

## Comment utiliser les catégories

### Pour rechercher

1. **Filtrer par catégorie** : Réduisez les résultats à un domaine
2. **Explorer** : Découvrez les compétences d'une catégorie
3. **Combiner** : Utilisez catégorie + mots-clés

### Pour votre profil

Quand vous ajoutez une compétence :

1. Choisissez la **catégorie** appropriée
2. Sélectionnez la **compétence** précise
3. Indiquez votre **niveau**

## Les niveaux de compétence

Indépendamment de la catégorie, chaque compétence a un niveau :

| Niveau | Description |
|--------|-------------|
| **Débutant** | Bases acquises, peut initier |
| **Intermédiaire** | Bonne maîtrise, peut enseigner les fondamentaux |
| **Expert** | Maîtrise avancée, peut aborder les aspects complexes |

### Importance du niveau

Le niveau aide à :

- **Trouver le bon partenaire** : Un débutant cherche plutôt un intermédiaire ou expert
- **Définir les attentes** : Ce qu'on peut apprendre avec ce membre
- **Équilibrer l'échange** : Niveaux similaires = échange plus équitable

## Évolution des catégories

### Pourquoi les catégories évoluent

Les catégories peuvent s'enrichir :

- Nouvelles compétences populaires
- Demandes de la communauté
- Évolution des domaines (ex: nouvelles technologies)

### Comment suggérer une compétence

Si une compétence manque :

1. Vérifiez qu'elle n'existe pas sous un autre nom
2. Identifiez la catégorie appropriée
3. Contactez l'équipe SkillSwap via le support

## En résumé

Les catégories sont l'ossature qui organise la diversité des compétences sur SkillSwap. Elles facilitent la recherche pour les apprenants et la visibilité pour ceux qui partagent leurs savoirs.
