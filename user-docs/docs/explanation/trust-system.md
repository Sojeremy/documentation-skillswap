# Le système de confiance

La confiance est au cœur de SkillSwap. Cette page explique comment elle est construite et maintenue au sein de la communauté.

## Pourquoi la confiance est essentielle

### Le défi des plateformes d'échange

Sur une plateforme où des inconnus se rencontrent, plusieurs questions se posent :

- Comment savoir si un membre est fiable ?
- Comment évaluer la qualité d'une compétence annoncée ?
- Comment se protéger des mauvaises expériences ?

### La solution SkillSwap

SkillSwap utilise plusieurs mécanismes pour établir la confiance :

1. **Les profils détaillés** - Informations vérifiables
2. **Les évaluations** - Retours de la communauté
3. **L'historique** - Trace des échanges passés

## Les évaluations

### Comment ça fonctionne

Après un échange, chaque participant peut évaluer l'autre :

| Élément | Description |
|---------|-------------|
| **Note** | De 1 à 5 étoiles |
| **Commentaire** | Texte libre (optionnel) |
| **Visibilité** | Publique sur le profil |

### Ce qui est évalué

Les membres évaluent généralement :

- **La qualité de l'enseignement** : Clarté, pédagogie, patience
- **La fiabilité** : Ponctualité, respect des engagements
- **La communication** : Réactivité, clarté des échanges
- **L'équité** : Équilibre de l'échange

### L'impact des évaluations

Les évaluations influencent :

- **La visibilité** : Les profils bien notés apparaissent en premier
- **La confiance** : Les nouveaux membres consultent les avis
- **La réputation** : Construction d'une identité sur la plateforme

## La note moyenne

### Calcul

La note moyenne affichée sur un profil est calculée à partir de toutes les évaluations reçues :

```
Note moyenne = Somme des notes / Nombre d'évaluations
```

### Interprétation

| Note moyenne | Interprétation |
|--------------|----------------|
| 4.5 - 5.0 | Excellent |
| 4.0 - 4.4 | Très bien |
| 3.5 - 3.9 | Bien |
| 3.0 - 3.4 | Correct |
| < 3.0 | À améliorer |

### Nombre d'évaluations

Le nombre d'évaluations est aussi important que la note :

- **Peu d'évaluations** (1-5) : La note peut ne pas être représentative
- **Évaluations modérées** (6-20) : Tendance fiable
- **Nombreuses évaluations** (20+) : Note très représentative

## Les profils

### Informations de confiance

Un profil complet inspire plus confiance :

- **Photo de profil** : Humanise le contact
- **Bio détaillée** : Montre l'investissement
- **Compétences précises** : Indique le sérieux
- **Disponibilités** : Prouve l'organisation

### Les nouveaux membres

Les membres sans évaluation ne sont pas moins dignes de confiance :

- Tout le monde commence à zéro
- Le profil complet compense l'absence d'historique
- Les premiers échanges construisent la réputation

## Les bonnes pratiques

### Pour établir la confiance

En tant que membre, vous pouvez :

1. **Compléter votre profil** au maximum
2. **Être précis** sur vos compétences et niveaux
3. **Respecter vos engagements** (horaires, contenu)
4. **Communiquer clairement** avant et après les échanges
5. **Évaluer systématiquement** vos partenaires

### Pour évaluer de manière constructive

Quand vous évaluez :

- Soyez **honnête** mais **bienveillant**
- Mentionnez les **points positifs**
- Formulez les critiques de manière **constructive**
- Évitez les évaluations de **représailles**

## La modération

### Cas problématiques

En cas de comportement inapproprié :

- Les évaluations très négatives alertent la communauté
- Les membres peuvent signaler un comportement
- L'équipe SkillSwap peut intervenir

### Protection des données

Les évaluations sont :

- **Publiques** : Visibles par tous les membres
- **Permanentes** : Non modifiables après envoi
- **Liées** : Associées à un échange réel

## En résumé

Le système de confiance de SkillSwap repose sur la transparence et la réciprocité. Chaque membre contribue à la qualité de la communauté en évaluant honnêtement ses échanges et en maintenant un profil à jour.
